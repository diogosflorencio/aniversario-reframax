# Gerador de Aniversariantes

Automatiza a criação da lista de aniversariantes a partir de planilhas Excel. Começou como script e evoluiu para um app com interface. Versão **v1.0.0**.

Funcionalidades:

- escolha de planilha, mês, layout do cartaz e estilo da lista
- layouts configuráveis, cada um com foto de fundo e posição de nomes e dias
- geração, salvamento e impressão da imagem
- preferências salvas
- bandeja do sistema ao fechar a janela
- lembrete mensal e geração/impressão automática

Pastas: `planilhas/` (dados), `modelos/` (fundos), `fontes/` (tipografia), `layouts/` (configuração), `outputs/` (imagens geradas).
